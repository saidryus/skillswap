Business Model Canvas
SkillSwap: Peer-to-Peer Academic Tutoring Platform
Key Partners
  • Philippine colleges and universities
  • Registrar offices (schedule & grade data)
  • University IT/infrastructure teams
  • Academic departments
Key Activities
  • System development & maintenance
  • Tutor application verification
  • Student schedule encoding
  • Platform monitoring & bug fixes
Key Resources
  • MERN stack expertise
  • MongoDB database
  • Institutional class schedule data
  • Server infrastructure
  • Tesseract.js (OCR)
Value Propositions
  • Automated conflict-free scheduling via constraint engine
  • Verified competency-based tutor matching
  • Reduced coordination overhead for peer tutoring
  • Structured peer learning platform
  • OCR-assisted grade verification
  • Visual timetable comparison
Customer Relationships
  • Self-service platform with admin oversight
  • Automated in-app notifications
  • Rating-based quality assurance
  • Admin-managed onboarding
Channels
  • Web application (browser-based)
  • University intranet/network
  • Institutional announcements
  • Social media (awareness)
Customer Segments
  • College students needing academic help (tutees)
  • Students with strong grades (potential tutors)
  • University administrators
  • Academic coordinators
Cost Structure
  • Development time (labor)
  • Server hosting
  • Domain registration (optional)
  • Maintenance overhead
Revenue Streams
  • Institutional subscription (annual SaaS license per HEI)
  • One-time implementation & setup fee per institution
  • Technical support & maintenance contracts
